# cheap-llm-router

**Экономьте токены подписки Claude или Codex — пусть более дешёвая модель читает файлы вместо дорогой.**

🌐 **Языки:** [English](README.md) · [Українська](README.uk.md) · **Русский**

> 🤖 **Не хотите устанавливать вручную?** Передайте этот промпт своему AI-агенту (Claude Code, Codex CLI, Cursor, Gemini, Aider — любому, кто умеет читать URL и выполнять shell-команды):
>
> > *«Прочитай <https://github.com/Lexus2016/cheap-llm-router> и выполни раздел* ***For AI agents (auto-install)*** *из README, чтобы установить, настроить, проверить и подтвердить, что `cheap` работает на этой машине. Спроси меня только о моём ключе OpenRouter API, когда он понадобится»*.
>
> Агент сам установит через pipx, попросит ключ один раз, добавит правило в ваши `CLAUDE.md` / `AGENTS.md`, проведёт smoke-test и отчитается. Проверено с Claude Code, OpenAI Codex CLI, Gemini CLI и Cursor.

## Что это делает, простыми словами

Когда Claude Code или OpenAI Codex CLI открывает для вас файлы, каждая их строка попадает в вашу дорогую модель, за которую вы платите подпиской. Прочитать пять файлов — это уже около 8 000 токенов (единиц текста, которые считает модель). Несколько таких операций в день — и вы упираетесь в недельный лимит.

`cheap-llm-router` добавляет одну небольшую команду — **`cheap`**. Она отправляет ваши файлы в значительно более дешёвую модель (DeepSeek, Kimi, Gemini Flash — на ваш выбор) и возвращает короткое резюме, обычно около 600 токенов. Ваша дорогая модель получает **ответ**, а не сырой текст файлов.

То же самое для истории чата: когда вы вот-вот сделаете «компакт» (сжатие диалога) длинной сессии или хотите обновить документацию по тому, что только что делали, команда `cheap extract` сама сделает резюме сессии — и дорогой модели не придётся перечитывать его целиком.

## Зачем это может быть нужно

- **Реже упираться в недельный лимит.** Или не упираться вовсе.
- **Тот же рабочий процесс.** Вы продолжаете работать в Claude или Codex; `cheap` подключается через одно правило в файле `CLAUDE.md` / `AGENTS.md`.
- **Любая дешёвая модель.** Работает со всем, что совместимо с OpenAI API. По умолчанию — `deepseek/deepseek-chat-v3-0324` (~$0.005 за типичный 4-файловый вызов, контекст 164K токенов) — проверенная non-reasoning модель, прошедшая наш регрессионный тест на 17 моделях. См. [Как выбрать модель](#как-выбрать-модель) — полная таблица.
- **Поддерживаются две программы.** Claude Code и OpenAI Codex CLI, с автоматическим распознаванием, в какой вы сейчас работаете.
- **Честен в ограничениях.** Не поможет в отладке, аудите безопасности или задачах, где нужен точный текст — там он говорит дорогой модели читать файлы самой.

### Что это значит для недельного лимита подписки

Claude Pro/Max и Codex Plus считают использование в **недельных окнах токенов**, не только в долларах. Уменьшение в 13 раз токенов на чтение — это примерно **разница между «упёрся в лимит в среду» и «упёрся когда-то в следующем месяце»**. Если вы регулярно бьётесь о недельный лимит, этот фикс — самый сильный по влиянию без изменения вашего рабочего процесса.

Дорогая модель продолжает думать. Дешёвая лишь делает механическую часть «прочитай эти файлы и скажи что в них», где глубокий разум не нужен.

## Быстрый старт (3 шага)

### 1. Установить

```bash
pipx install git+https://github.com/Lexus2016/cheap-llm-router.git
```

(Берём `pipx`, а не `pip`, чтобы у программы было своё изолированное Python-окружение. На macOS: `brew install pipx`.)

### 2. Задать ключ OpenRouter

Зарегистрируйтесь бесплатно на <https://openrouter.ai/>, пополните несколько долларов, скопируйте ваш ключ, затем:

```bash
echo 'export OPENROUTER_API_KEY=sk-or-v1-...' >> ~/.zshrc
source ~/.zshrc
cheap config check       # → OK
```

### 3. Сообщить Claude / Codex о новом инструменте

```bash
cheap install-claude-rule
```

Готово. Теперь, когда Claude или Codex собирается прочитать несколько файлов просто чтобы понять контекст, она автоматически использует `cheap`.

### Обновление правила позже

Если вы заново запустите `cheap install-claude-rule` после нового релиза, команда останется тихой, когда локальное правило уже актуально. Если вышла более новая версия правила — увидите подсказку:

```
already installed at /Users/you/.claude/CLAUDE.md (rule v1; v2 available
  — run with --force to upgrade, this REPLACES the block including any
  local edits)
```

`--force` перезаписывает весь блок `## Cheap LLM …`, так что любые ваши локальные правки в нём (например, собственные записи в «Past mistakes») будут потеряны — скопируйте их отдельно заранее, если хотите сохранить.

### Опционально: PreToolUse hook (только Claude Code)

Правило само по себе полагается на то, что агент сам вспомнит о делегации перед каждым `Read`. На длинных сессиях эта дисциплина падает ниже 5%. Опциональный hook закрывает разрыв тем, что **в момент** вызова `Read` добавляет подсказку прямо в контекст агента — но только когда чтение действительно delegate-worthy (полный файл ≥200 строк или ≥2 полных чтения за несколько последних turn'ов). Остальные Reads (короткие файлы, line-targeted, после Edit на этот же файл, secrets, изображения) проходят без шума.

```bash
cheap install-hook        # добавляет PreToolUse:Read запись в ~/.claude/settings.json
```

Hook никогда не блокирует: всегда возвращает `permissionDecision: "allow"` и добавляет только `additionalContext` как подсказку. Другие агенты (Codex CLI, Cursor, Aider, Cline, Continue, OpenCode, Gemini CLI) пока не имеют такого же hook-API — для них единственным механизмом делегации остаётся правило из `cheap install-rule`.

## For AI agents (auto-install)

Если ты AI-агент (Claude Code, Codex CLI, Gemini, Cursor и т.п.), читающий этот README чтобы установить `cheap` для своего пользователя — выполни эти шаги по порядку. Остановись и спроси пользователя, если для какого-то шага не имеешь нужной информации.

```bash
# 1. Убедиться что pipx установлен (пример для macOS):
command -v pipx >/dev/null 2>&1 || brew install pipx
pipx ensurepath

# 2. Установить (или принудительно обновить) cheap-llm-router:
pipx install --force git+https://github.com/Lexus2016/cheap-llm-router.git

# 3. Проверить статус — сообщит об отсутствии OPENROUTER_API_KEY:
cheap config check
```

Если `cheap config check` сообщает об отсутствующем `OPENROUTER_API_KEY`:

1. **ОСТАНОВИСЬ и спроси пользователя** его ключ OpenRouter API — он создаёт его на <https://openrouter.ai/keys>.
2. Добавь в профиль shell (рекомендуется — секрет не попадает в файл под git):
   ```bash
   echo 'export OPENROUTER_API_KEY=sk-or-v1-...' >> ~/.zshrc
   source ~/.zshrc
   ```
   Для изолированных окружений — вместо этого редактируй `~/.config/cheap-llm/config.yaml`, закомментируй строку `# api_key_env:` и установи `provider.api_key: "sk-or-v1-..."`.

Затем продолжай:

```bash
# 4. Установить правило в ~/.claude/CLAUDE.md и ~/.codex/AGENTS.md:
cheap install-claude-rule

# 5. Smoke-test — должен напечатать markdown-резюме, НЕ пустой/короткий вывод:
cheap read README.md -q "что это за проект?"
```

**Проверь прежде чем сказать пользователю «готово»:**

- `cheap config check` возвращает `OK`.
- `cheap config show` показывает `model: deepseek/deepseek-chat-v3-0324` (или другую проверенную non-reasoning модель — см. [Как выбрать модель](#как-выбрать-модель)).
- Smoke-test возвращает непустой контент (>500 байт на stdout).

**Если smoke-test возвращает пустой / очень короткий вывод**, настроенная модель является reasoning-способной (`deepseek/deepseek-v4-pro`, `kimi-k2.6`, `kimi-k2-thinking`, `glm-5.1`, `glm-4.7-flash`, `mimo-v2.5`) — она потратила весь cap `max_summary_tokens` на скрытые thinking-токены, не оставив видимого вывода. Измени строку `model:` в `~/.config/cheap-llm/config.yaml` на проверенную non-reasoning модель и повтори smoke-test.

**Прежде чем брать новую модель за дефолт**, запусти регрессионный скрипт что ловит все три известных режима провала (пустой вывод, сломанный cap, потеря файлов):

```bash
git clone https://github.com/Lexus2016/cheap-llm-router.git /tmp/clr
/tmp/clr/scripts/test-model.sh <provider/model-id>
```

Скрипт автоматически делает бэкап конфига, запускает single-file + 4-file тесты, печатает вердикт PASS / NOT RECOMMENDED с причинами, и восстанавливает оригинальный конфиг при выходе (даже на Ctrl-C).

## Две основные команды

### `cheap read FILE…`

Делает резюме списка файлов. Используйте, когда у вас есть **вопрос** о коде, а не задача изменить его.

```bash
cheap read src/auth.py src/db.py src/api.py -q "как тут устроен вход?"
```

Вы получите резюме примерно на 600 токенов в формате markdown, плюс одну строку телеметрии (отчёт о вызове), где чётко написано, сколько это стоило:

```
[cheap] files=3 input_chars=15823 output_tokens=587 model=deepseek/deepseek-chat-v3-0324 elapsed_ms=2143
```

### `cheap extract`

Делает резюме **истории текущей сессии чата**. Полезнее всего прямо перед `/compact` в Claude или когда вы хотите обновить документацию по тому, что только что делали.

```bash
cheap extract -q "что мы сегодня решили?"
```

Программа сама понимает, в Claude Code вы или в Codex CLI, и находит нужный файл сессии. Передайте `--mode messages-only`, чтобы оставить только сообщения (без шума от служебных вызовов инструментов), или `--tail 50`, чтобы оставить только 50 последних сообщений.

## Где живёт API-ключ

Два поддерживаемых варианта. Выберите один.

| Где лежит ключ | Когда использовать |
|---|---|
| **Переменная окружения** `OPENROUTER_API_KEY` | **Рекомендуется.** Ключ не попадает ни в один файл. |
| **Поле `api_key:`** в `~/.config/cheap-llm/config.yaml` | Изолированные оболочки, GUI-запускалки, где переменные окружения неудобны. |

Если заданы оба, побеждает тот, что в YAML-файле (явное указание в приоритете).

Когда ключ лежит в YAML, `cheap config show` автоматически маскирует его как `***REDACTED***`. Но `cat $(cheap config path)` этого не делает — поэтому держите файл подальше от git, синхронизации iCloud / Dropbox и скриншотов.

## Работает с любым AI-кодинг-агентом — дорогая модель думает, дешёвая читает

Ваша подписочная модель — Claude Opus, GPT-5, Sonnet, Gemini Pro — стоит вам недельной квоты токенов и когда рассуждает о коде, и когда просто читает 5 файлов для контекста. **Чтение — это механическая работа.** Нет смысла платить за неё по reasoning-ценам.

`cheap-llm-router` ставит модель за $0.20/M на вход перед вашей подпиской. Дешёвая берёт на себя «прочитай эти файлы — что там?» — а ваша подписка получает **ответ** (~600 токенов) и тратит свой ценный контекст на то, в чём она хороша: думать, решать, писать правильный код.

**Вы сохраняете:**
- Качество мышления подписки (Opus / GPT-5 / Gemini Pro дальше делают всю настоящую работу)
- Свой IDE / CLI / чат-процесс — `cheap` подключается через одно правило
- Каждый существующий промпт, правило проекта, привычку

**Вы избавляетесь от:**
- ~85% токенов на «прочитай эти файлы» рутину
- Большинства ударов о недельный лимит — получаете недели до cap'а
- Умственного напряжения экономить подписку

**Проверенные AI-агенты:** Claude Code · OpenAI Codex CLI · Cursor · Gemini CLI · Aider · Cline · Continue.dev · Plandex · Crush · Goose · OpenCode · aichat. Работает с чем угодно ещё, что читает `CLAUDE.md` / `AGENTS.md` файлы правил или может вызывать shell-команды.

> **Не платите зарплатой senior-архитектора за ксерокопирование документов.**

## Как выбрать модель

Не каждая дешёвая модель подходит. Три режима провала которых надо избегать:

1. **Reasoning-модели** (например `deepseek/deepseek-v4-pro`, `kimi-k2.6`, `kimi-k2-thinking`, `glm-5.1`, `glm-4.7-flash`, `mimo-v2.5`) — молча тратят весь cap `max_summary_tokens` на скрытые thinking-токены, возвращая пустой видимый output.
2. **Модели игнорирующие cap** (например `kimi-k2.5`, `z-ai/glm-4.7`) — превышают cap в 2-8 раз, съедая ваш token-бюджет.
3. **Модели теряющие покрытие** (например `kimi-k2`, `gpt-5.4-nano`) — фокусируются на первых 1-2 файлах и игнорируют остальные при multi-file чтениях.

### Проверенные модели (4-файловый тест 79K chars input, cap=1500 токенов, sweep 2026-05-05)

| Модель | Время | Tokens | Покрытие | Context | $/4-file | Заметки |
|---|---|---|---|---|---|---|
| **`deepseek/deepseek-chat-v3-0324`** ★ | **36с** | 1237 | **4/4** | 164K | $0.005 | DEFAULT — стабильная, быстрая |
| `google/gemma-4-31b-it` | 50с | 1500 | 4/4 | 262K | $0.003 | ~35% дешевле, чуть медленнее |
| `deepseek/deepseek-v4-flash` | 78с | 1500 | 4/4* | 1M | $0.003 | Для больших whole-module чтений (>200K chars). На retest'ах было 3/4 в 2 из 3 попыток — используй когда нужен контекст. |

\* `deepseek/deepseek-v4-flash` — используй только когда нужен её 1M контекст. Сочетай с `max_input_chars: 2500000` в `~/.config/cheap-llm/config.yaml`.

### Проверь любую новую модель перед адаптацией

Репозиторий содержит регрессионный скрипт что ловит все три режима провала:

```bash
git clone https://github.com/Lexus2016/cheap-llm-router.git /tmp/clr
/tmp/clr/scripts/test-model.sh openai/gpt-5.4-mini      # любой provider/model-id
```

Он делает бэкап конфига, запускает single-file + 4-file тесты, печатает вердикт PASS / NOT RECOMMENDED с причинами (пустой вывод, cap сломан, файлы потеряны, медленно), и восстанавливает конфиг при выходе — даже на Ctrl-C.

Стоимость одного запуска: примерно $0.005-$0.02 в зависимости от модели. Дешевле чем обнаружить сломанный default в проде.

## Частые вопросы

**Поймёт ли сама модель, когда использовать `cheap`, а когда нет?**
Да — `cheap install-claude-rule` записывает точный чек-лист в `CLAUDE.md` (и `AGENTS.md`). Правило говорит: *используй `cheap` только когда нужен **ответ**, а не точный текст. Не используй для редактирования, аудита безопасности или всего, где нужно цитировать или сравнивать строки точно*.

**Безопасно ли это с секретами?**
`cheap read` по умолчанию отказывается отправлять файлы, чьи имена попадают под шаблоны `.env*`, `*.key`, `*.pem`, `id_rsa`, `credentials.json` и т. п. Можно переопределить через `--include-sensitive` (с предупреждением), но лучше так не делать.

**Куда реально уходят мои данные?**
В OpenRouter, а оттуда — к выбранному провайдеру (DeepSeek, Moonshot, Google…). Если вы уже пользуетесь Claude или Codex, ваш код и так проходит через одного стороннего AI-провайдера. Этот инструмент добавляет ещё одного, но качественно картину не меняет.

**Сколько реально экономится?**
8× дешевле чем скармливать файлы Claude Opus 4.7 напрямую, 6× против Sonnet 4.6, ~3× против GPT-5-codex. ~$1.65 на одного активного разработчика в день на ценах Opus 4.7, ~$45/мес. См. [Реальные цифры](#реальные-цифры--насколько-это-удешевляет-работу) выше — полная таблица. `cheap` пишет одну строку телеметрии на каждый вызов, так что можно проверить математику на собственном рабочем процессе.

**Я не пользуюсь Claude. Можно взять это с любой оболочкой?**
Да — `cheap read` и `cheap extract` — обычные shell-команды. Запускайте вручную, перенаправляйте результаты, делайте что хотите. Шаг `install-claude-rule` необязательный.
